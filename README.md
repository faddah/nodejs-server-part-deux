### To Run —

At your command prompt, first run —

```
$ npm install
```

...to install the `node_modules` folder and all the npm dependencies. Then run —

```
$ npm start
```

...and then go to your browser and load [http://127.0.0.1:3000](http://127.0.0.1:3000).
